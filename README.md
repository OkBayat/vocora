# واژه‌یار IELTS

یک جعبه لایتنر آفلاین برای تمرین املای لغات IELTS Listening. هیچ دیتابیسی ندارد و همه‌ی اطلاعات را در `localStorage` مرورگر ذخیره می‌کند.

## اجرا

فایل `index.html` را با Chrome، Edge یا Firefox باز کنید. فهرست ۱۵۰۰ لغت از قبل داخل برنامه قرار دارد.

برای اطمینان بیشتر از ذخیره‌شدن داده‌ها یا استفاده روی موبایل، می‌توانید پوشه را با یک وب‌سرور ساده باز کنید:

```bash
python -m http.server 8080
```

سپس آدرس `http://localhost:8080/leitner-ielts/` را باز کنید.

## پشتیبان‌گیری

- «پشتیبان کامل» تمام لغات، تنظیمات و تاریخچه را نگه می‌دارد.
- «خروجی برای تحلیل با ChatGPT» یک JSON خلاصه و امن از وضعیت یادگیری می‌سازد.
- حذف داده‌های مرورگر، استفاده از حالت ناشناس یا تغییر مرورگر می‌تواند داده‌ها را پاک کند؛ به‌طور منظم پشتیبان بگیرید.

## ورود فایل جدید

فایل Markdown یا TXT باید هر واژه را در یک خط شماره‌گذاری‌شده داشته باشد. تیترهای `##` به عنوان دسته‌بندی ذخیره می‌شوند:

```md
## Travel
1. accommodation
2. itinerary
3. centre / center
```

موارد جداشده با `/` به‌عنوان املای جایگزین پذیرفته می‌شوند.
